## Testing & Validation

**Default Testing Protocol**: All website validation and testing MUST be done using the Chrome DevTools MCP Server.

See [TESTING-PROTOCOL.md](TESTING-PROTOCOL.md) for complete testing guidelines and procedures.

**Quick Start Testing:**
1. Start Flask app: `python app.py`
2. Use MCP server with visible Chrome browser for all validation
3. Browser window should be visible during testing (never headless)
